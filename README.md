# Knowledge Compression Theory (KCT)

知识压缩理论白皮书 - 基于认知科学的框架化学习模型

## 简介

本理论提出"知识框架 = 压缩算法"的核心隐喻，学习目的 = 提高压缩强度。

## 核心框架

- **六步学习法**：架构层 → 基础层 → 专业层 → 转化层 → 应用层 → 互动层
- **双重维度**：压缩模型（领域）× 压缩比例（水平）
- **九大理论整合**：认知负荷、工作记忆、图式理论、建构主义、信息加工、双重编码、专家技能、遗忘曲线、SECI模型
